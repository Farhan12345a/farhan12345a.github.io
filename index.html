Projects

Project 1: Smart Medicine Dispenser

Senior Capstone Project incorporated myself and four other members to create a Smart Medicine Dispenser. Interactive webpages were initially created with HTML/CSS/Javascript. Created further interaction using ReactJS, AWS Services (Amplify, IoT Core, S3, DynamoDB).


Project 2: Farhan Fabrics Website

Custom HTML/CSS/Javascript website created from scratch. Was my first ever web development project and I decided to create a website that sold Christmas Clothes. Website provides user interaction such as creating/viewing reviews, checking out items, and contacting the company.


Project 3: Calorie Counter

Custom Full Stack Web Application created using the MERN stack (MongoDB, ExpressJS, ReactJS, and NodeJS). All of the data is stored and accessed using MongoDB Atlas. URL routing is implemented by using ExpressJS features and handling GET/POST requests.
